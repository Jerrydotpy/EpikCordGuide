# Installing

## Windows

```python
py -3 -m pip install -U Epikcord.py
```

## MacOS/Linux

```python
# Linux/macOS
python3 -m pip install -U "Epikcord.py"
```

## Development Version

```python
pip install git+https://github.com/Epikcord/Epikcord.py
```
